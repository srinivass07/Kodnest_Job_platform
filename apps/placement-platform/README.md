# Placement Readiness Platform

A modern web application for placement preparation featuring practice problems, mock interviews, and progress tracking.

## Features

### Landing Page
- Hero section with "Ace Your Placement" heading
- Features grid showcasing:
  - Practice Problems
  - Mock Interviews
  - Track Progress
- Professional footer

### Dashboard
- Sidebar navigation with 5 sections:
  - Dashboard (Home with statistics)
  - Practice (Coding problems)
  - Assessments (Timed tests)
  - Resources (Study materials)
  - Profile (User settings)
- Header with branding and user avatar
- Responsive design

## Technology Stack

- **HTML5**: Semantic structure
- **CSS3**: Custom properties, Flexbox, Grid
- **Vanilla JavaScript**: DOM manipulation, routing
- **Color Scheme**: Indigo/Purple (hsl(245, 58%, 51%))

## Usage

### Local Development
Simply open `index.html` in your browser. No build process or dependencies required.

### Deployment
This is a static site that can be deployed to:
- Vercel
- Netlify
- GitHub Pages
- Any static hosting service
